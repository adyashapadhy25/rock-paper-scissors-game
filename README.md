# Rock Paper Scissors Game

An interactive Rock Paper Scissors game built using **HTML, CSS, and JavaScript**.  
The game features a modern user interface, animations, and a live score counter between the player and the computer.

## Features
- Interactive gameplay
- Modern and responsive UI
- Hover animations
- Live score tracking
- Random computer choice

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run the Project
1. Download or clone the repository.
2. Open the project folder.
3. Double click **index.html**.
4. The game will open in your browser.
